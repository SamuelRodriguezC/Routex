# Routex
Sistema web para la optimización de rutas logísticas, diseñado para centralizar la información operativa, garantizar la calidad de los datos mediante validaciones y mejorar la toma de decisiones en la planificación y ejecución de procesos, con trazabilidad completa de las operaciones.


“Aunque la consigna menciona persistir únicamente registros válidos, se implementó persistencia de rutas inválidas con estado FAILED para garantizar trazabilidad y auditoría, alineado con el ejemplo proporcionado.
